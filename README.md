# ドラフト会議 Web アプリ (mdraft)

簡易的なドラフト会議をリアルタイムで行える Web アプリです。各チームが自分の端末から選手を指名し、重複時はサイコロ（自動）で獲得者を決定します。

## 仕組み

- 指名対象選手 40 名 / 参加チーム 9 / 全 4 巡
- 各巡で未確定のチームが選手を指名 → 全員そろうと一斉開票
- 単独指名 → 獲得確定
- 重複指名 → サイコロを自動で振り、最大の出目のチームが獲得（同値は振り直し）
- 敗れたチームは同じ巡で再指名。全チーム確定まで繰り返し
- 1〜4 巡すべて完了で終了

## 構成

- `server/` … Node.js + TypeScript + Express + Socket.IO（状態管理・進行ロジック）
- `client/` … React + Vite + TypeScript（UI）

## セットアップと起動

2 つのターミナルで実行します。

### 1. サーバー

```powershell
cd server
npm install
npm run dev
```

`http://localhost:3001` で起動します。

### 2. クライアント

```powershell
cd client
npm install
npm run dev
```

`http://localhost:5173` をブラウザで開きます。

> クライアントを別マシンから使う場合は、サーバーの URL を環境変数 `VITE_SERVER_URL` で指定してください（例: `client/.env` に `VITE_SERVER_URL=http://<サーバーIP>:3001`）。

## 使い方

1. 進行役の端末で「進行役（ホスト）になる」を選択し、チーム名・選手名を入力して「設定を保存」。
2. 各チームは自分の端末でアプリを開き、担当チームを選んで参加。
3. 進行役が「ドラフト開始」。
4. 各チームが選手を指名 → 自動開票 → 進行役が「次へ」で進行。
5. 4 巡完了で結果を表示。
